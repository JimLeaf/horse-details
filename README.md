# Horse Details UI

A front-end project with searchable dropdowns, unique horse selection, and image buttons.

## Setup
1. Clone the repo
2. Open `index.html` in your browser
3. Add icons to `images/` for Track, Distance, Style

## Features
- Searchable dropdowns (powered by Select2)
- Unique horse selection enforced
- Modular project structure (HTML, CSS, JS, images)
